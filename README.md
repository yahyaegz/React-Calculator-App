# React-Calculator-App

A responsive calculator built with React that supports basic arithmetic operations, keyboard input, and a modern glassmorphism user interface. The app uses `useReducer` for state management and is designed to provide a clean and interactive user experience.

## Features
- Basic arithmetic operations (addition, subtraction, multiplication, division)
- Keyboard support
- Responsive design with Glassmorphism
- State management with `useReducer`

## Getting Started

### Prerequisites
- Node.js

### Installation
1. Clone the repository
2. Run `npm install`
3. Run `npm run dev` to start the development server
