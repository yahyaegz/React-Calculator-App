## 🚀 Overview

This is a modern **React Calculator App** built with a clean and interactive **glassmorphism UI**. It supports **basic arithmetic operations**, **keyboard input**, and uses **`useReducer`** for efficient state management.

This project demonstrates:
- Component-based React development
- Reducer-based state management
- Keyboard event handling
- Responsive and modern UI styling

---

## 🎯 Features

- ✅ Perform basic arithmetic operations
- ✅ Responsive glassmorphism-inspired design
- ✅ Keyboard input support
- ✅ Clear and delete functionality
- ✅ Built with React and `useReducer`
- ✅ Clean and interactive user experience

---

## 🖼️ UI Highlights

- Soft glassmorphism design
- Clean calculator layout
- Easy-to-use button arrangement
- Supports both mouse and keyboard interaction
